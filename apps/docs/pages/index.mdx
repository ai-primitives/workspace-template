# Introduction

Welcome to the documentation site. This site demonstrates the integration of a documentation app within the monorepo structure.

## Features

- Built with Next.js and Nextra
- Integrated with workspace configurations
- Uses shared components from the monorepo
- Supports MDX for rich content
